üöÄ Quick Start Guide

Step 1: Test solar output analysis

curl -X POST "/solar/output" -d '{"latitude": 40.7, "longitude": -74.0, "system_capacity": 5.0}'

Step 2: Create futures contracts

curl -X POST "/futures/electricity" -d '{"latitude": 40.7, "longitude": -74.0, "system_capacity_kw": 5000}'

Step 3: Export results for analysis

curl -X POST "/futures/electricity-csv" -d '{"latitude": 40.7, "longitude": -74.0, "system_capacity_kw": 5000}'

üí° Integration Examples

Python Example:

import requests

# Basic solar analysis
response = requests.post("https://chainfly-derivative-finance.onrender.com/solar/output", 
    json={"latitude": 40.7, "longitude": -74.0, "system_capacity": 5.0})
solar_data = response.json()

# Futures with bifacial enhancement
response = requests.post("https://chainfly-derivative-finance.onrender.com/futures/electricity",
    json={
        "latitude": 40.7, "longitude": -74.0, "system_capacity_kw": 5000,
        "bifaciality": 0.8, "albedo": 0.7
    })
futures_data = response.json()

JavaScript Example:

// Solar output analysis
const solarResponse = await fetch('/solar/output', {

});
const solarData = await solarResponse.json();

// Futures pricing
const futuresResponse = await fetch('/futures/electricity', {
    method: 'POST', 
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({
        latitude: 40.7, longitude: -74.0, system_capacity_kw: 5000,
        bifaciality: 0.8, albedo: 0.7
    })
});
